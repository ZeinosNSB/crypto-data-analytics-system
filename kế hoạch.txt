TUẦN 1: Khởi tạo & Thống nhất
--------------------------------------------------------------------------------
[CA TEAM] : Họp chốt "Data Contract". Thống nhất cấu trúc file JSON đẩy vào 
            Kafka (Vd: {symbol, price, volume, timestamp}). Đăng ký GitHub.
[DevOps]  : Dựng file docker-compose.yaml v1 (Kafka, Zookeeper, Redis, MongoDB).
            Đẩy lên Git để cả team clone về chạy local.
[Ingest]  : Tạo project Python, cài CCXT, thử kết nối API Binance in ra console.
[Batch]   : Khởi tạo project PySpark, viết script đọc file JSON mẫu (Mock data).
[Stream]  : Khởi tạo project Spark Streaming, viết hàm đếm số dòng từ console.
[Web]     : Khởi tạo project Next.js và FastAPI. Vẽ mockup giao diện ra giấy.

--------------------------------------------------------------------------------
TUẦN 2: Thông luồng cơ bản - Local
--------------------------------------------------------------------------------
[Ingest]  : Hoàn thiện Kafka Producer. Đẩy data Binance thật vào topic Kafka.
[DevOps]  : Cài đặt Kafka UI để team nhìn thấy luồng tin nhắn. Nghiên cứu K3s.
[Batch]   : Chuyển sang đọc data từ topic Kafka (local) -> Ghi ra file Parquet.
[Stream]  : Đọc topic Kafka -> In nến 1 phút (OHLC) ra màn hình console.
[Web]     : Viết API GET (Mock data) trên FastAPI. Next.js gọi API hiển thị.
*** MILESTONE 1: Dữ liệu thật từ Binance phải nhảy được trên terminal của Spark.

--------------------------------------------------------------------------------
TUẦN 3: Xử lý cốt lõi
--------------------------------------------------------------------------------
[DevOps]  : Thêm MinIO vào docker-compose. Hỗ trợ Batch setup đường dẫn S3.
[Batch]   : Ghi data từ Kafka xuống MinIO. Viết Job tính ROI hoặc Volume 24h.
[Stream]  : Cấu hình output, đẩy nến 1 phút và cảnh báo giá trực tiếp vào Redis.
[Ingest]  : Code thêm cơ chế tự động reconnect nếu rớt mạng Websocket Binance.
[Web]     : FastAPI kết nối thành công Redis. Hiện nến thật (chưa real-time).

--------------------------------------------------------------------------------
TUẦN 4: ghép hệ thống
--------------------------------------------------------------------------------
[Batch]   : Ghi kết quả tính toán phức tạp từ PySpark vào MongoDB.
[Web]     : FastAPI kết nối MongoDB. Next.js hiện biểu đồ lịch sử & chỉ số.
[Cả Team] : Chạy thử luồng: Binance -> Kafka -> Spark -> DB -> FastAPI -> Web.
            Sửa lỗi Schema, đồng bộ múi giờ (Timezone UTC/Local).
*** MILESTONE 2: Hệ thống chạy thông suốt trên Docker Compose. Đạt điểm B+.

--------------------------------------------------------------------------------
TUẦN 5: A+ FEATURES (Tính năng nâng cao)
--------------------------------------------------------------------------------
[Batch]   : Thêm 1 thuật toán MLlib nhỏ (phát hiện bất thường) hoặc tính RSI.
[Stream]  : Viết logic "Whale Alert" (cảnh báo lệnh lớn) đẩy vào Redis.
[Web]     : Dùng WebSockets kết nối Next.js và FastAPI để giá nhảy tự động (Real-time).
[DevOps]  : Dựng xong Minikube/K3s trên máy. Viết Helm values.yaml cho Kafka.
[Ingest]  : Tối ưu script để lấy 5-10 cặp coin cùng lúc thay vì 1 cặp.

--------------------------------------------------------------------------------
TUẦN 6: KUBERNETES DEPLOYMENT
--------------------------------------------------------------------------------
[DevOps]  : Deploy Kafka, MinIO, Redis, Mongo lên K8s. Cài Resource Limits.
[Cả Team] : Đóng gói code thành Docker Image. Tạo file YAML deploy lên K8s.
            Phối hợp với DevOps sửa các lỗi không kết nối được mạng nội bộ Pod.
*** LƯU Ý: Nếu K8s sập hoặc ăn hết RAM, lùi về Docker Compose để kịp tiến độ.

--------------------------------------------------------------------------------
TUẦN 7: MONITORING & TUNING (Giám sát & Tối ưu)
--------------------------------------------------------------------------------
[DevOps]  : Cài Prometheus + Grafana. Vẽ dashboard đo RAM, CPU, Kafka Lag.
[Batch]   : Tối ưu bộ nhớ Spark (Checkpointing) để tránh lỗi Out Of Memory.
[Stream]  : Cấu hình Watermarking để xử lý data đến trễ.
[Web]     : Tối ưu UI/UX, CSS, xử lý lỗi vặt trên giao diện Next.js.
[Ingest]  : Chạy test ngắt mạng đột ngột xem script có tự hồi phục không.

--------------------------------------------------------------------------------
